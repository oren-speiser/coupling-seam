Stability Diagnostics Instrument 02 Coupling Seam

Where a machine is weakest is not inside a part. It is the seam between parts.

Most condition monitoring reads one domain at a time: pressure, heat, power, each alone. But many failures form in the handoff between domains, where load in one quietly raises load in the next. From raw multi-domain sensor data, with no prior knowledge of the machine, this instrument locates the weak seam and the dominant failure chain. The same method, unchanged, run on two different real machines below.

One governance law, Θ = I/R (driving load over absorption capacity), two questions. Instrument 01 asks when a machine will fail (cross-domain early warning ↗). This one asks where the weak seam is.

Strength and danger are not the same axis

A coupling matters when the system's least-stable mode runs through it, not simply when it is large. Sensitivity is the dominant-eigenvector weight on each edge: the standard measure of how much it drives instability.

From diagnosis to action

Does a targeted repair beat spreading the same effort?

Same repair budget, four ways to spend it. Margin gain is how much the coupling load ρ is reduced. Targeting is guided by the dominant-eigenvector sensitivity; the alternatives spread the budget or chase the largest coupling. This is where the two machines differ, honestly.

What this is, exactly

Scope, plainly stated

Data
Real, public benchmarks. Hydraulic rig: UCI Condition Monitoring of Hydraulic Systems (Helwig et al., 2015), a physical test rig, 2,205 cycles, 17 sensors. Turbofan: NASA C-MAPSS FD001 (Saxena & Goebel, 2008), 100 engines run to failure.
Method
Each sensor reduced to one value per cycle and standardized; sensors grouped into physical domains; coupling estimated as partial correlation across the cycles, isolating direct domain-to-domain dependence. Identical method on both machines, nothing re-tuned.
Math
Established, not new. Dominant-eigenvalue stability, eigenvector-guided edge selection (Restrepo–Ott–Hunt; Van Mieghem et al.), Perron–Frobenius. The contribution is the cross-domain framing and its application to real multi-domain data.
Honest limit
The diagnostic reading transfers across both machines. The targeted-repair advantage does not: it is real where coupling is uneven (the hydraulic rig) and vanishes where the danger is a near-uniform cluster (the engine). Partial correlation is observed dependence, not proven causation.
